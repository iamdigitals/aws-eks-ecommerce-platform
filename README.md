# AWS EKS E-Commerce Platform

A cloud-native e-commerce application deployed on Amazon EKS.

## Architecture

Users
 ↓
Route53
 ↓
Application Load Balancer
 ↓
Amazon EKS
 ↓
Frontend Pods
 ↓
Backend Pods
 ↓
Amazon RDS
